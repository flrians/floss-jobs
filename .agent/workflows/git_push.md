# Git Push Workflow

This workflow automates the process of saving your work and pushing it to the remote repository.

// turbo
1. Stage all changes:
```bash
git add .
```

2. Commit changes (you can customize this message when the system prompts you if you prefer):
```bash
git commit -m "Update Floss Jobs project: New changes to the codebase"
```

// turbo
3. Push to the main branch:
```bash
git push origin main
```
